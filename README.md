# fr-tv-replay
Debug in progress, no release for now.....

Search &amp; play several French Television Replay streams (Pluzz, Arte, Canal+) - Python / CLI

This application will list the streams available on several French TV channels :
 * Pluzz
 * Arte
 * Canal+
